# umi-template

> 利用 `umi` 脚手架的一个前端模板，该模板集成了日常使用的 `utils` 和一些基础模块如：登录模块。

## 版本

> v2.0.0

## 更新内容

1. 引入 `typescript` 作为辅助开发语言。
2. 引入第三方 `@umi/hook`。
3. 引入 `mockjs` 模拟 API 数据。
4. 对第三方库 `axios` 进行封装优化。
5. 集成内部鉴权服务判断, `Auth-service`。
6. 新增内部变量。
7. 对本地存储进行项目名区分。
